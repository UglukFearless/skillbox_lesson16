This is home work for lesson 16 skillbox school "Redux".

Простая программа, предназначенная для отработки создания веб-приложения с применением React и Redux.
Является модификацией предыдущего приложения, выполненного для домашнего задания React.

Использованы следующие модули:
+ react
+ react-dom
+ react-redux
+ redux

+ webpack
+ babel-core
+ babel-loader
+ babel-preset-es2015
+ babel-preset-react

Программа позволяет добавлять и удалять комментарии. Комментарии сохраняются в локальном хранилище.
